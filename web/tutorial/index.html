This framework will allow you to click on a selected color button and call the specific sets of slides of the color's value. This framework could also be used for non-color buttons or slides. The following steps will teach you how to utilize this framework for creating buttons that will call different colored slideshows.

Step Two: Declare HTML Color and Control Buttons. These will be the clickable buttons to change slideshows or the slides in a slideshow. 
            button class="btn1" onclick="changeColor('Platinum'), changeSlide(0)">Platinum Silver
            button class="btn2" onclick="changeColor('Burgundy'), changeSlide(0)">Burgundy Red
            button class="btn3" onclick="changeColor('Cobalt'), changeSlide(0)">Cobalt Blue
            button class="btn4" onclick="changeColor('Gold'), changeSlide(0)">Graphite Gold

            button class="prev" onclick="changeSlide(-1)">Previous Slide
            button class="play" onclick="startSlideShow()">Play
            button class="next" onclick="changeSlide(1);">Next Slide

Step Three: Declare a HTML Slideshow Container div. This is where your slideshow will be held. Fill it with the first image you want to be seen when the web page loads. For this tutorial, two slideshow container divs will be shown to represent two different objects are being returned from the framework. 

            div class = slideshow-container >

            img src ="../pics/tutorialpics/Surface-Laptop-Plat1.jpg" id="slideshow2" />

Step Four: Declare the JavaScript slideshow objects and slideshow functions. The slideshow object will take in the slideshow's reference id, starting slideshow pictures for when the web page loads, and the color of the slideshow. The changeSlide function will go to the next or previous slide on button click. The startSlideShow function will make the slideshow play automatically. Modify the changeColor function and colorArray to your desired colors and slideshow pictures. Note: The slideshowObj.changeColor(slides) is commented because the tutorialFw.js also has a commented out function if you want to place your pictures in the framework itself.

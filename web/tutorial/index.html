This framework will allow you to click on a selected color button and call the specific sets of slides of the color's value. This framework could also be used for non-color buttons or slides. The following steps will teach you how to utilize this framework for creating buttons that will call different colored slideshows.

Step Two: Declare HTML Color and Control Buttons. These will be the clickable buttons to change slideshowObj's color or the nested slides in that slideshow's chosen color. 

            div class = "ColorButtons"> Color Buttons:
        button class="btn1" onclick="slideshowObj.setColor('Platinum'), slideshowObj.changeSlide(0)">Platinum Silver /button>
        button class="btn2" onclick="slideshowObj.setColor('Burgundy'), slideshowObj.changeSlide(0)">Burgundy Red /button>
        button class="btn3" onclick="slideshowObj.setColor('Cobalt'), slideshowObj.changeSlide(0)">Cobalt Blue /button>
        button class="btn4" onclick="slideshowObj.setColor('Gold'), slideshowObj.changeSlide(0)">Graphite Gold /button>
        /div>
        
        div class = "ControlButtons"> Control Buttons:
        button class="prev" onclick="slideshowObj.changeSlide(-1)">Previous Slide /button>
        button class="next" onclick="slideshowObj.changeSlide(1);">Next Slide /button>
        /div>

Step Four: Declare the JavaScript slideshow objects and slideshow functions. First, the slideshow object will take in the slideshow's reference id to know where to place the slides. Second, the slideshow object will take in a slideIndex initialized at 0, so each slideshow will have their own index if you choose to have multiple slideshows. Third, the slideshow object will take in the slidePics array, which will have the colors with the a nested array of the slides associated with the chosen color. Thus, when the color button is clicked or the color is chosen, the framework will iterate and produce a slideshow based on nested array's colored pictures. 


            var slideshowObj = slideShowFW.makeSlideShow({
                id: "slideshow",
                slideIndex: 0,
                slidePics: [
                    {"color": "Platinum", "slides": ["../pics/tutorialpics/Surface-Laptop-Plat1.jpg", "../pics/tutorialpics/Surface-Laptop-Plat2.jpg", "../pics/tutorialpics/Surface-Laptop-Plat3.jpg", "../pics/tutorialpics/Surface-Laptop-Plat4.jpg", "../pics/tutorialpics/Surface-Laptop-Plat5.jpg"] },
